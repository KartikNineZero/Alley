Installing Alley

Alley is a fast, free web browser. Before you download, you can check if Alley supports your operating system and you have all the other system requirements.

Desktop

  • Install Alley on Windows
    1. Download the installation file.
    2. If prompted, click Run or Save.
    3. If you chose Save, double-click the download to start installing.
    4. Windows 7: An Alley window opens once everything is done.
    5. Windows 8 & 8.1: A welcome dialog appears. Click Next to select your default browser.
    6. Windows 10: An Alley window opens after everything is done.
